发生多态要有两个前提条件：  

1：继承。多态发生一定是子类和父类之间。  
2：重写。子类重写了父类的方法。  

例如：

    class Figure:
        def draw(self):
            print('绘制几何图形...')

    class Eclipse(Figure):
        def draw(self):
            print('绘制椭圆图形...')

    class Triangle(Figure):
        def draw(self):
            print('绘制三角图形...')

    f1 = Figure()
    f1.draw()

    f2 = Eclipse()
    f2.draw()

    f3 = Triangle()
    f3.draw()

    绘制几何图形...
    绘制椭圆图形...
    绘制三角图形...

与Java等静态语言相比，多态性对于动态语言Python而言意义不大。多态性优势在于运动期动态特性。在Java中多态性是指，编译期声明变量是父类的类型，在运行期确定变量所引用的实例。而Python不需要声明变量的类型，没有编译，直接由解释器运行，运行期确定变量所引用的实例。
